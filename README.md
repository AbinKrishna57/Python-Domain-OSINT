# PYTHON DOMAIN OSINT



This is a python tool that is used to find the publicly available information on a domain

Ex: DNS Records, Whois Information, IP Information and Subdomains



### Required Packages



pip install dnspython

pip install ipwhois

pip install knockpy

pip install httpx\[http2]

pip install bs4

pip install pyOpenSSL

pip install dotenv



#### To install all the packages required

pip install -r requirements.txt
